# Product Roadmap Template

Copy this into each product repo as `PRODUCT_ROADMAP.md`.

## Product Goal
- One clear sentence describing the target outcome.

## Definition Of Done
- [ ] Must-have features are complete.
- [ ] No open critical reliability issues.
- [ ] UI/UX pass completed.
- [ ] Core verification commands pass.
- [ ] Documentation updated.

## Milestones
- M1 Foundation
- M2 Core Features
- M3 Bug Fixing And Refactor
- M4 UI/UX Improvement
- M5 Stabilization And Release Readiness

## Current Milestone
- M1 Foundation

## Brainstorming Queue
- Candidate ideas not yet selected (keep this list deep and aligned).

## Pending Features
- [ ] Feature 1
- [ ] Feature 2

## Delivered Features
- YYYY-MM-DD | Feature | Evidence | Commit

## Bugs And Refactor Queue
- [ ] Bug fix
- [ ] Refactor

## UI/UX Improvements
- [ ] Improvement item

## Verification
- Command | Status | Notes

## Risks And Blockers
- Risk | Impact | Mitigation

## Next Cycle Plan
- Top 3 priorities for the next iteration.

## Drift Checks
- Is current work aligned to product goal and milestone?
- If not, what should be de-prioritized immediately?
